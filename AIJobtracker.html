AI-Powered Outlook Job Tracker

An advanced job application tracking system that leverages AI and real-time notifications to help users stay on top of job applications from their Outlook emails.

    Project Features
    • Intelligent Company Detection - Automatically detects company websites and provides real-time application status notifications.
    • AI-Powered Analytics - Supports natural language querying, statistical analysis, and company-specific insights with vector similarity search for precise company matching.
    • Automated Data Management - Regularly processes emails, categorizes data, and maintains up-to-date vector embeddings.
    Tech Stack
    • Backend & AI - FastAPI framework in Python with PostgreSQL (pgvector) and GPT-4 for intelligent querying and data orchestration using Prefect.
    • Frontend - Chrome Extension with JavaScript, HTML, and CSS, featuring a custom popup UI and a natural language query interface.
    • Infrastructure & DevOps - Hosted on AWS (EC2, RDS), containerized with Docker & Docker Compose, CI/CD enabled via GitHub Actions, and version controlled with Git.
